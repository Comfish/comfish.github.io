2019目标

发表于 2019-01-03

用2个词来形容我的2018，那应该是「自信」和「焦虑」。淡化回顾，给2019定几个小目标。

2019目标

  1. 开始学车（指标：是否有报名）
  2. 英语记录生活（指标：英文日记52篇）
  3. 成为一名Vloger（指标：B站粉丝1000个）
  4. 出国旅游1次（备注：用钱就能解决的目标 2333）
  5. 达成6块腹肌（备注：有就可以，不需要大）
  6. 寻找商业价值（指标：以非工资的形式赚到5000RMB）

定制年度目标的习惯有4年了，但完成度都不超过30%。为了提高目标的完成度，我决定参考公司的考核方式，每个季度进行一次绩效考核。
